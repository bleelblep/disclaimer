02 catalog entry
file · glupo-001 private collection

glupo is what this whole thing was before it had a name. a junk drawer for every glyph idea that didn't warrant its own app — and a few that eventually did.

the bingsu theme system (ten palettes, inspired by korean shaved-ice desserts) lives here. the glyph matrix helpers live here. every toy is a floor plan i didn't want to throw away.

it isn't listed, isn't signed, isn't supported. if a toy gets good enough, it moves out and gets its own page. everything else just hangs around keeping the wiring warm.
