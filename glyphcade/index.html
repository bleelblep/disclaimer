WHAT IS IT arcade_cabinet.apk

GlyphCade turns the nothing phone 3's glyph matrix into a proper little arcade cabinet. it boots with a coin drop, shows a pixel-art selection screen, and then you're playing — all of it rendered on the led grid, not the main display.

tilt, tap, and swipe. scores persist. the phone screen stays off while you play. it's an arcade that fits in the palm of your hand and doesn't ask for much.

FEATURES inserts.txt

  • rendered entirely on the glyph matrix — main screen stays dark while you play
  • tilt + touch controls, per-game sensitivity
  • sound effects + background music, toggleable per game
  • haptic feedback, because arcade cabinets vibrate
  • CRT scanlines, coin drop animation, countdown timer
  • high scores saved between sessions
  • easy-mode toggles for the meaner games

REQUIREMENTS hardware.cfg

  • nothing phone 3 — for the glyph matrix (no matrix, no arcade)
  • permission to use the glyph matrix + accelerometer
  • a steady hand for the tilt games
  • some tolerance for experimental software
▲ heads up ▲

this app is very experimental

things may break. games may crash mid-high-score. the coin drop animation may loop if it feels like it. it's v0.1 — expect weirdness and be gentle with your save file.
