thinking of you

open source · nothing phone 3

two devices. one small pattern. no words.

there's no message. there's no notification.
you just know.

what it is (｡•ᴗ•｡)

a quiet little thing that pairs two nothing phone 3s and lets them share a small led pattern in real-time. when one person updates the pattern, the other's glyph matrix reflects the change.

that's it. no chat, no presence indicator, no notification layer. just shared light.

how it works ⸜(｡˃ ᵕ ˂ )⸝

1
two devices pair using a shared code — this code becomes a firebase database key
2
both devices listen on the same path: /pairs/{code}/pattern
3
when one side writes a pattern, the other renders it on the glyph matrix instantly
4
last write wins — no history, no read receipts, no confirmation

setup ٩(ˊᗜˋ*)و

this isn't on the play store. it's an open-source project you build yourself:

  1. create a firebase project with realtime database enabled (free tier is fine)
  2. configure database rules to restrict read/write to paired devices
  3. add your firebase api key to gradle.properties
  4. build and sideload the apk via android studio
  5. pair with a friend using the same code ♡

what to know ( ˃ ⌑ ˂ )

  • nothing phone 3 only for the native glyph matrix experience
  • other devices render patterns as an on-screen overlay (not the intended feel)
  • older nothing phones use different glyph systems — would need custom mapping
  • pattern data is stored as plaintext in firebase; anyone with console access can see it
  • no accounts, no hosted backend beyond firebase
heads up

this project is not maintained. it's an open-source release — no ongoing bug fixes, no updates, no play store.

feel free to fork it, remix it, or just read the code. the point was always that it exists, not that it's polished.
