Fill in the details below and click Open GitHub Issue. A pre-filled issue will open in a new tab — submit it and a maintainer will add your pack to the registry.

How it works

  1. Create your pack

    Follow the Custom Packs Guide to build your pack.json and animations folder, then ZIP everything up.

  2. Host it on GitHub Releases

    Create a GitHub repository, make a Release, and upload your ZIP file. Copy the download URL for the next step.

  3. Fill the form above

    Submit the form — it opens a pre-filled GitHub Issue. A maintainer reviews it and adds your pack to the registry, usually within a day or two.
