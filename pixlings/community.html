How to Submit Your Pack

  1. Create your pack following the Custom Packs Guide
  2. Host it on GitHub with a release (or use any direct download link)
  3. Fork the pixlings-docs repo
  4. Add your pack info to packs.json
  5. Add a preview image (PNG or GIF) to images/packs/
  6. Submit a Pull Request!
